# 更新日志 (Changelog)

## 2026-02-14 (项目审计与规划)
- **[文档] 项目审计**: 完成了全面的项目代码审计，生成了 `project_audit.md`。
- **[发现]** 识别出以下关键问题：
    - 缺失根目录 `requirements.txt`。
    - 存在硬编码的敏感信息 (Redis 密码)。
    - 缺少项目启动文档 (`README.md`)。
    - 存在新旧两套 Controller (`gui` vs `drission`)。
- **[规划]** 确定了后续优化路线图：文档补全 -> 配置解耦 -> 代码清理。

## 2026-02-13 (爬虫管理与详情页重构)
- **[功能] 爬虫管理增强**: 在 Admin 后台实现了爬虫任务的**启动、暂停、恢复、停止**功能。
- **[架构] 详情页爬虫重构**:
    - 建立了 Spider、Controller 和 Database 之间的严格串行同步机制。
    - 解决了并发导致的重复采集问题。
    - 优化了 URL 参数解析，统一使用 `urllib.parse`。
- **[优化]** 增加了 15s 强制跳过机制，防止浏览器端死锁。

## 2026-02-07 (爬虫核心优化)
- **[核心] 去重机制升级**: 引入 **Redis Bloom Filter** 替换数据库级去重，显著提升过滤效率。
- **[同步] 滚动同步**: 实现了 `hasMore` 状态的精准同步，当列表加载完毕时自动停止滚动。
- **[数据] 上下文补全**: 修复了职位数据缺失行业/城市信息的问题，支持从 URL 参数中回填。
- **[兼容性]** 适配 Scrapy 2.14+，修复了废弃 API 警告。

## 2026-01-26 (简历智能化与监控)
- **[特性] 智能简历**: 实现了 PDF 上传 -> AI 解析 -> 自动生成并填充简历的全流程。
- **[监控] Celery 日志**: 集成 `TaskLog` 模型，自动记录后台任务的执行状态与耗时。
- **[体验] 无感刷新**: 实现了 Token 过期自动刷新机制 (Silent Refresh)。
